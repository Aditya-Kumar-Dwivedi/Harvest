# Harvest
It is a basic shopping app which shows the products available and involves some basic inputs from the User .
It is bulid using react native.In addition to it I have also gained the knowledge about calling and posting info through apis while making this app

# Prerequities of the app 

To run this app on any system the system should have the following depedencies installed:
* An andriod device connected or and andriod simulator installed on your system 
* expo-CLI installed on your system.

# How to kickstart the app
* Fork this repository to your own or download the code and open it in VS code or whatever ide you prefer.
* open the terminal and run the following command
```
npm build
```
it will install the node modules which are required for the app
* Then run the following command to run the app
```
expo start
```
